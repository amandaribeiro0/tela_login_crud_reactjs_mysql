# tela_login_crud_reactjs_mysql

# Installation

# Client
1. $ cd client
2. $ npm install 
  or $ yarn install
3. $ npm start 
 or $ yarn start

# Server
1. Change your mySQL database data server/index.js
2. $ cd server
3. $ npm install 
  or $ yarn
4. $ npm rum devStart



# Used Libraries
# Front-end
* @material-ui
* Axios
* Back-end
* express
